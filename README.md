# davidademuyiwa.com

Static personal academic website for David Ademuyiwa.

## Pages

- `index.html` - profile
- `publications.html` - publications
- `experience.html` - experience

## Local Preview

Open `index.html` directly in a browser, or run a local static server from this folder:

```powershell
python -m http.server 8000 --bind 127.0.0.1
```

## GitHub Pages

This folder is ready for GitHub Pages. The `CNAME` file points the site at `davidademuyiwa.com`, and `.nojekyll` disables Jekyll processing.
